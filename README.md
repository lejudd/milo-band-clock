# milo-band-clock

## Overview:

The Milo Champions Band was targeted towards kids as a clock / step counter. 
The band is no longer supported along with the app being discontinued.
This code aims to bring some life back to the tiny device, primarily allowing it to continue functioning as a clock.
YMMV

## Running:

`python3 milo_time.py`
